## 技术

#### [Introducing NextUI Version 2.0](https://nextui.org/blog/nextui-v2)

NextUI 是一个与 Vercel 风格很类似的 React 的组件库，不过它与 Vercel 没有任何关系。近期发布的 2.0 版本进行了比较大的更新，CSS 方案从 [Stitches](https://stitches.dev/) 切换到了 [TailwindCSS](https://tailwindcss.com/)，并支持 React 服务端组件，大部分组件基于 [react-aria](https://react-spectrum.adobe.com/react-aria/) 实现，并且大量使用了 [framer-motion](https://www.framer.com/motion/)，动画效果比较灵动。

#### [Twine by SwiftGG—学习 Apple 技术的好帮手](https://mp.weixin.qq.com/s/kWbDdzJap09II8xTtosjlg)

Twine 是 SwiftGG 翻译组发布的一款浏览器插件，支持 Safari 和 Chrome 浏览器。它的灵感来源于[沉浸式翻译](https://immersivetranslate.com/)，可以实现苹果官方开发文档的页面在原有英文内容基础上显示中文翻译，方便不熟悉英文的人阅读文档。不同于沉浸式翻译的机器翻译，Twine 的翻译内容全部由人工翻译，内容托管在 GitHub。

## 开源

#### [anime.js](https://github.com/juliangarnier/anime)

anime.js 是一个轻量的 JavaScript 动画库，API 简单易用，官网上展示的效果也都比较惊艳。它与 framer-motion 比较类似，但 anime.js 不限制任何框架。

#### [tailwind-variants](https://github.com/nextui-org/tailwind-variants)

这里的变种 (variants) 是指一个组件的不同形式，比如按钮组件有实心、Ghost、有边框等多种形式。tailwind-variants 的作用是组合 TailwindCSS 的类，来方便地实现组件的多种形式，其灵感来源于 Stitches，并且支持 TypeScript，框架无关。

#### [Hydrogen](https://github.com/AirCodeLabs/Hydrogen)

[AirCode](https://aircode.io/) 开源的 Serverless 框架。与传统的 Node.js 框架 (如 Express 等) 不同的是，它只需要简单地创建一个文件即可实现一个云函数，同时内置了数据库能力和文件 API。
