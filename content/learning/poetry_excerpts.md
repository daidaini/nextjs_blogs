## 好句摘抄
```txt
世路如今已惯，此心到处悠然
---- 张孝祥《西江月》

人生只似风前絮，欢也零星。
悲也零星。都作连江点点萍。
---- 王国维 《采桑子》

夜闲须尽醉，莫负百年心。
---- 王昌龄《杂曲歌辞·少年行二首》

世间行乐亦如此，古来往事东流水。
---- 李白 《梦游天姥吟留别》

人生一世间，忽若暮春草。
---- 徐干《室思》

一朝去金马，飘落成飞蓬。
---- 李白《相和歌辞·东武吟》
```

```txt
《唐多令 芦叶满汀洲》
----  刘过 (宋)
芦叶满汀洲，寒沙带浅流。二十年重过南楼。柳下系船尤未稳，能几日，又中秋。
黄鹤断矶头，古人今何在？旧江山浑是新愁。欲买桂花同载酒，终不似，少年游。
```

```txt
《一剪梅 舟过吴江》
---- 蒋捷 (宋)
一片春愁待酒浇。江上舟摇，楼上帘招。秋娘渡与泰娘桥，风又飘飘，雨又潇潇。
何日归家洗客袍？银字笙调，心字香烧。流光容易把人抛，红了樱桃，绿了芭蕉。
```


```txt
浮云游子意，落日故人情
---- 李白《送友人》

仍怜故乡水，万里送行舟
---- 李白《渡荆门送别》

飞鸟没何处，青山空向人
---- 刘长卿《送留王维》

山回路转不见君，雪上空留马行处
---- 岑参《送武判官归京》

孤帆远影碧空尽，唯见长江天际流
---- 李白《送孟浩然之广陵》

衰兰送客咸阳道，天若有情天亦老
---- 李贺《金铜仙人辞汉歌》

猿啼客散暮江头，人自伤心水自流
---- 刘长卿《重送裴郎中贬吉州》

谁谓波澜才一水，已觉山川是两乡
---- 王勃《秋江送别二首其一》

惊起归鸿不成字，辞柯落叶最知秋。
---- 黄庭坚 《次韵任君官舍秋雨》

晓来风，夜来雨，晚来烟。 
是他酿就春色，又断送流年。
---- 张惠言《水调歌头 春日赋示杨生子掞》

不知乘月几人归，落月摇情满江树。
---- 张若虚《春江花月夜》

斜阳外，寒鸦万点，流水绕孤村。
---- 秦观《满庭芳 山抹微云》
```

```txt
不拟人间更求事，些些疏懒亦何妨？

此时情绪此时天，无事小神仙。

寻常风月，等闲谈笑，称意即相宜。

是非得丧皆闲事，休向南柯与梦争。

闲听天籁静看云，心境俱清。

小舟从此逝，江海寄余生。

山中莫道无供给，明月清风不用钱。

须信百年皆是梦，天地阔，且徜徉。

且酩酊，任他两轮日月，来往如梭。
```


## 苏轼
```txt
《定风波》
莫听穿林打叶声
何妨吟啸且徐行
竹杖芒鞋轻胜马
谁怕
一蓑烟雨任平生

料峭春风吹酒醒
微冷
山头斜照却相迎
回首向来萧瑟处
归去
也无风雨也无晴
```

```txt
《念奴娇-赤壁怀古》
大江东去， 浪淘尽， 千古人物。
故垒西边，人道是，三国周郎赤壁。
乱石穿空，惊涛拍岸，卷起千堆雪。
江山如画，一时多少豪杰！

遥想公瑾当年，小乔初嫁了，
雄姿英发，羽扇纶巾。
谈笑间，樯橹灰飞烟灭。
故国神游，多情应笑我，早生华发。
人生如梦，一樽还酹江月。
```

```txt
《水龙吟 次韵章行质夫杨花词》
似花还似非花，也无人惜从教坠。
抛家旁路，思量却是。
萦损柔肠，困酣娇眼，欲开还闭。
梦随风万里，寻郎去处，又还被莺呼起。

不恨此花飞尽，恨西园，落花难缀。
晓来雨过，遗踪何在？一池萍碎。
春色三分，二分尘土，一分流水。
细看来，不是杨花，点点是离人泪。
```


```txt
《江城子 密州出猎》
老夫聊发少年狂，左牵黄，右擎苍。
锦帽貂裘，千骑卷平冈。
为报倾城随太守，亲射虎，看孙郎。

酒酣胸胆尚开张，鬓微霜，又何妨！
持节云中，何日遣冯唐？
会挽雕弓如满月，西北望，射天狼。
```


```txt
《江城子 乙卯正月二日夜记梦》
十年生死两茫茫，不思量，自难忘。
千里孤坟，无处话凄凉。
纵使相逢应不识，尘满面，鬓如霜。

夜来幽梦忽还乡，小轩窗，正梳妆。
相顾无言，惟有泪千行。
料得年年肠断处，明月夜，短松冈。
```


```txt
《水调歌头 快哉亭作》
落日绣帘卷，亭下水连空。
知君为我，新作窗户湿青红。
长记平山堂上，欹枕江南烟雨，渺渺没孤鸿。
认得醉翁语，山色有无中。

一千顷，都镜净，倒碧峰。
忽然浪起，掀舞一叶白头翁。
堪笑兰台公子，未解庄生天籁，刚道有雌雄。
一点浩然气，千里快哉风。
```

```txt
《卜算子 黄州定慧院寓居作》
缺月挂疏桐，漏断人初静。
谁见幽人独往来，缥缈没孤鸿。
惊起却回头，有恨无人省。
拣尽寒枝不肯栖，寂寞沙洲冷。
```


```txt
《蝶恋花 春景》
花褪残红青杏小。
燕子飞时，绿水人家绕。
枝上柳绵吹又少，天涯何处无芳草。

墙里秋千墙外道。
墙外行人，墙里佳人笑。
笑渐不闻声渐悄，多情却被无情恼。
```
